Installation

Get started with TermuxVoid in just a few simple steps.

# Add TermuxVoid repository to your Termux installation
curl -sL https://termuxvoid.github.io/repo/install.sh | bash
# Update package lists after adding the repo
pkg update
# Install any tool from the repository
pkg install <tool-name>

Frequently Asked Questions

Common questions from the Alienkrishn legacy repository.

Is TermuxVoid safe to use?

All packages are maintained by the same trusted team behind Alienkrishn's repository. We prioritize security and stability in all our packages.

How often are packages updated?

We provide weekly updates for all packages, plus urgent security patches as needed. Our maintenance schedule is more regular than the original repository.

Can I request new tools to be added?

Yes! Message @nullxvoid on Telegram with your tool requests. We evaluate all suggestions based on community interest and technical feasibility.

What's the license for these packages?

TermuxVoid maintains the same BSD 3-Clause license as the original Alienkrishn repository. You can view the full license text in our GitHub repository.
