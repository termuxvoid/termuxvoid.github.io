◆ What is TermuxVoid?

Unofficial custom APT repository for Termux. Provides extra tools not available in official Termux repos or excluded from it. Covers ethical hacking, bug bounty, cyber security, reverse engineering & more. Compiled natively for Android. Includes root packages — see root-repo.md.

▶ Quick Start

Get started in two simple commands:

# Add repository
curl -sL https://termuxvoid.github.io/repo/install.sh | bash
# Install any tool
pkg install <tool-name>
# Example
pkg install metasploit-framework

All tools are for legal security research and ethical hacking purposes only. Always obtain proper authorization before testing systems you do not own.

Can I request new tools?

Yes! Submit requests via:

How do I report a broken package?
